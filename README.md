# ATM_MANAGEMENT_SYSTEM

The ATM Management System is a console-based Python application that simulates the basic working of an Automated Teller Machine (ATM). This project is designed for students and beginners to understand core programming concepts such as functions, conditional statements, loops, and OOPS concepts.

  It allows users to securely perform banking operations like checking balance, withdrawing money, depositing money, and changing PIN.

  
**OBJECTIVES:**

To understand real-time banking operations logic

To practice Python programming fundamentals

To improve problem-solving and logical thinking

**FEATURES:**

*    Creation of the bank account
*    Set a pin for security purpose
*    Balance Enquiry if you enter the correct pin
*    Cash withdrawl
*    cash deposit
*    show the account details
*    bank history
  
**HOW IT WORKS?**

*   User creates a bank account
*   Set a pin
*   menu is displayed
*   user selects require banking operation
*   user enters an account number and pin
*   Transaction is processed
*   again menu is displayed until the user selects exit choice

  **How to Run the Project:**

1. Install Python (version 3 or above)


2. Download or clone this repository


3. Open terminal or command prompt


4. Navigate to the project folder


5. Run the program using:

python atm.py


**Conclusion:**

This ATM Management System project is a simple yet effective way to understand how real-world banking applications work using Python. It strengthens core programming concepts and serves as a strong foundation for advanced projects.
